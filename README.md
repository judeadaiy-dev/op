# 🚀 تطبيق محادثة احترافي - Flutter + Supabase

تطبيق محادثة كامل مبني بـ Flutter و Supabase. يدعم الثيم الفاتح/الداكن، إشعارات Realtime، رفع الصور، بحث، لوحة تحكم أدمن، وتصميم Glassmorphism.

## ✨ المميزات

- ✅ **تسجيل دخول**: إيميل + باسورد + Google
- ✅ **محادثات Realtime**: رسائل فورية عبر Supabase
- ✅ **رفع الصور**: في المحادثات والبروفايل
- ✅ **بحث**: في المستخدمين والرسائل
- ✅ **إشعارات**: عداد غير مقروء + Realtime
- ✅ **لوحة أدمن**: إحصائيات + إدارة مستخدمين + إعدادات التطبيق
- ✅ **ثيم احترافي**: فاتح/داكن تلقائي + Glassmorphism
- ✅ **دعم عربي**: RTL كامل + خط Cairo
